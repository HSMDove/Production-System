I want to build a project that helps me with my work for the Tech Voice channel. I want you to create a simple but powerful web application that works as my personal platform for organizing content, tracking news, and turning everything into ready-to-use video ideas. I am a tech content creator who works on multiple series, and my main problem is that sources are scattered everywhere and ideas get lost. I want this system to become my external brain for content organization.

I want the app to be built around folders. For example, I can create a folder called “Android” or “Apple” or “Artificial Intelligence” or anything else. Each folder represents a topic or content category. Inside each folder I should be able to add sources related to that specific field, like news websites, YouTube channels, X accounts, or RSS feeds. For every source I only need a name, the link, and the type. In the background I want the system to fetch the latest content from these sources on a schedule like every hour or so, and store it in a database. Use RSS when available or simple scraping if needed.

When I open any folder, I want to see a “feed” of the latest news coming from all the sources inside that folder. The feed should show things like the title, a short summary generated by AI, the source name, the original link, and the publication date. I want to be able to filter inside the folder, for example by source type or by newest first. The most important thing is that browsing the feed should feel fast and clear.

The main part of the project is the “Generate Ideas” button. I want a clear button inside each folder that says “Generate ideas”. When I press it, the system should take the latest content inside that folder from the last three days by default, analyze it using AI, and return a list of video ideas suitable for Tech Voice in my style. The generated ideas must be in Arabic and follow the content formats I actually use: Thalathiyat (three-item episodes), “Leh?”, “Tech Injection”, “Device Tricks”, “App Tricks”, “Quick News”, and short-form content. Each idea should come with a suggested title, the series it fits best, a short explanation of the idea, the news links that triggered the idea, a suggested angle (storytelling, comparison, analysis, leaks etc), and an importance rating from 1 to 5 so I can know which ones are worth producing.

Every generated idea should be saved automatically into an “Ideas” table inside the app. Each idea should have a workflow state, starting as “raw idea”, and I can manually move it to “needs research”, “ready for script writing”, “script in progress”, “ready for filming”, “completed”. I want a dedicated Ideas page where I can see all ideas in a table view and also in a kanban board view sorted by status. For each idea I need to be able to add manual notes, attach a script link, attach a thumbnail link, and attach a montage text link if I want to connect the idea to Notion or an external document.

For the interface, I want the design to be clean and visually appealing. The entire layout must fully support Arabic and right-to-left UI because all my content is in Arabic and I want the experience to look natural and smooth.

The application itself does not have to be complicated or overly designed. The core experience is that from the main dashboard I can see the folders, enter any folder, browse its news, and easily generate ideas. The smart part of the system should happen in the backend when analyzing the news and producing the ideas. If the system needs an AI API key, prepare a simple settings page where I can paste my key.

In short, I want a personal content platform built around folders, each with its own sources. The system fetches the latest news, lets me browse them, and with one button converts the last few days of news into structured, categorized, Arabic video ideas tailored for Tech Voice. The platform should also include a simple workflow system so each idea can move from “raw idea” all the way to “ready to publish”.
